# Dotfiles do Caio2203

Repositório de configurações pessoais (dotfiles) usado para personalizar e configurar o ambiente de desenvolvimento. Este repositório contém configurações para **Hypr**, **Waybar** e **Zsh**.

---

## Estrutura do Repositório

- `hypr/`  
  Contém arquivos de configuração para o **Hyprland** (gerenciador de janelas Wayland).

- `waybar/`  
  Agrupa configurações para o **Waybar**, barra de status para ambientes Wayland.

- `zsh/`  
  Inclui configurações para o shell **Zsh**, com dotfiles como `.zshrc`, abreviações, temas ou plugins.
